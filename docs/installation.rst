============
Installation
============


Stable release
--------------

To install API-client-generic, run this command in your terminal:

.. code-block:: console

    $ pip install api_client_generic

This is the preferred method to install API-client-generic, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for API-client-generic can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/AETH-erial/api_client_generic

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/AETH-erial/api_client_generic/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/AETH-erial/api_client_generic
.. _tarball: https://github.com/AETH-erial/api_client_generic/tarball/master
